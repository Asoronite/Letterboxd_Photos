# Privacy Policy — Letterboxd Photos

**Last updated:** May 2025

## Data collection

Letterboxd Photos does not collect, store, transmit, or share any personal data.

## API key storage

The TMDB API Read Access Token you enter is stored exclusively in your browser's local Chrome storage (`chrome.storage.local`). It never leaves your device except to authenticate requests made directly to the TMDB API (`api.themoviedb.org`).

## Third-party services

This extension fetches image data from [TMDB (The Movie Database)](https://www.themoviedb.org/). Requests are made directly from your browser to TMDB's API. TMDB's own [privacy policy](https://www.themoviedb.org/privacy-policy) applies to those requests.

## Analytics and tracking

None. The extension contains no analytics, tracking scripts, or telemetry of any kind.

## Contact

This is a hobby project with no commercial goals. For questions or concerns, open an issue on [GitHub](https://github.com/Asoronite/Letterboxd_Photos).
