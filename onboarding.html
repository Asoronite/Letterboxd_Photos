Get your free TMDB key

Six small steps. The form in step 4 trips most people up — we've spelled out exactly what to put in each field so it doesn't.

  1. 1

    Create a free TMDB account

    Sign up with an email at themoviedb.org/signup and verify the confirmation email they send.

  2. 2

    Open the API settings page

    Once logged in, go to themoviedb.org/settings/api (or click your avatar → Settings → API in the left sidebar).

  3. 3

    Request a new key

    Click “Click here” next to Request an API Key, then pick Developer (free) and accept the terms.

    Pick Developer — not Personal. Personal is reserved for commercial partnerships and asks for a lot more info.

  4. 4

    Fill out the application form

    This is where most people get stuck. The fields ask developer-y questions for a personal-use case — anything reasonable works:

    Type of Use Website
    Application Name Personal use
    Application URL http://localhost (anything works)
    Application Summary Personal use to display TMDB photos on my browser.

    The summary field has a minimum length — the line above is long enough. The rest of the form (name, country, etc.) is straightforward. Approval is usually instant.

  5. 5

    Copy the Read Access Token

    After approval, the API page shows two values. You want the API Read Access Token — the long one starting with eyJ…. Click the copy icon next to it.

    Don't copy the short API Key (v3 auth) at the top — the extension uses bearer authentication, which needs the longer Read Access Token instead.

  6. 6

    Paste it into the extension

    Click the Letterboxd Photos icon in your browser toolbar (if you don't see it, pin it from the puzzle-piece menu), paste the token into the field, and hit Save & Validate.

That's it

Open any letterboxd.com film or person page and you should see the gallery appear in the sidebar. You can revisit this guide any time from the extension popup — there's a Setup guide link at the bottom.
